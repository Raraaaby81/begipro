< >span</>
部分的に編集する
spanを使うことで、文章の途中や一部分を改行せずに、見た目を変えることができます。
重ねて使うこともできます。
使用場所
body内部
使用例
<span> ... </span>
htmlファイル
使用する要素 : span  h1  h2
  1. <h1>spanを使うと文章の<span><span class="claspan">一部分</span>だけ</span>編集することができます。<span>便利！</span></h1>
cssファイル
使用する要素 : color  font-size
  1. span{
  2.   color: red;
  3. }
  4. .claspan{
  5.   font-size: 3vw;
  6. }

spanを使うと文章の一部分だけ編集することができます。便利！
